# Asset Guidelines

## Image Standards
- Use WebP wherever possible
- Compress before upload
- Max recommended image size: 300KB
- Hero images: under 500KB

## Naming Convention
Use lowercase kebab-case.

Example:
- groom-portrait.webp
- wedding-stage.webp

## Gallery Standards
- Portrait and landscape versions should be optimized separately
- Avoid raw DSLR uploads

## Fonts
- Self-host fonts when possible
- Keep total font payload minimal

## Audio
- Optional background audio should remain under 2MB

## Favicon
- SVG preferred
