# Animation System

## Philosophy
Animations should feel elegant, calm, and premium.

Avoid:
- aggressive motion
- bouncing effects
- flashy startup-style transitions

## Standard Timing
- Fast: 200ms
- Medium: 400ms
- Slow: 700ms

## Easing
Use smooth ease-in-out curves.

## Allowed Motion Types
- fade-up
- blur reveal
- stagger fade
- slow parallax
- opacity transitions

## Scroll Triggers
- Trigger once
- Avoid repetitive animation replay

## Mobile Optimization
- Reduce heavy transforms on low-end devices
- Avoid excessive blur on mobile GPUs
