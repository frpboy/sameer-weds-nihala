# UI/UX & Design Guidelines

## Style Direction
Modern Islamic luxury minimal

## Palette
- Warm ivory
- Champagne gold
- Deep emerald
- Soft beige
- Matte black typography

## Typography
**Headings:**
- Cinzel
- Cormorant Garamond

**Body:**
- Poppins
- Inter

*(This combination works best for premium wedding aesthetics.)*

## Spacing System
- Ensure breathable section spacing.
- Standardized container widths for readability.

## Motion Rules
- Fade timings: Slow and deliberate.
- Hover behavior: Subtle opacity or slight scale changes.
- Scroll behavior: Smooth scrolling.

## Components
- **Buttons**: Elegant borders, minimal backgrounds.
- **Cards**: Soft shadows, clean borders.
- **Countdown Blocks**: Legible numbers, subtle entry animations.
- **Gallery Layouts**: Masonry or clean grid.
