# Product Requirements Document

## Product Name
Shabin Weds Sana

## Vision
Create an elegant premium wedding invitation microsite optimized for mobile sharing and emotional engagement.

## Primary Goal
Deliver a memorable digital invitation experience.

## Target Users
- Family members
- Friends
- Business associates
- Elder relatives

## Core Features
- Animated intro (Splash Screen)
- Hero section with couple names and date
- Countdown
- Event Timeline / Wedding details
- Venue + Maps integration
- Gallery Placeholder
- RSVP
- WhatsApp & Instagram sharing

## Success Criteria
- Opens instantly on mobile
- Easy WhatsApp & Instagram sharing
- Elegant UI
- Stable deployment
