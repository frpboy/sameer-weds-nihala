# Design System

## Colors
- **Primary**: `#C7A97F` (Champagne gold)
- **Secondary**: `#F8F4EE` (Warm ivory / Soft beige)
- **Accent**: `#2E4A3D` (Deep emerald)
- **Text**: `#1A1A1A` (Matte black)
- **Romantic Mocha**: `#6F5B57` (Poetic quotes tone)
- **Muted Emerald**: `#355C4D` (Atmospheric particles tone)

## Typography Hierarchy
- **H1 / Couple Names**: `Cinzel` / `Cormorant Garamond`. Rendered as clean single-line `firstName` formatting (`"Mohammed Sameer"` & `"Nihala Jasmin"`) for perfect symmetry.
- **H2, H3 / Titles**: `Cinzel` / `Cormorant Garamond` with ceremonial tracking (`tracking-widest`).
- **Poetic Quotes**: `Cormorant Infant`, italic, `#6F5B57`, `0.88 opacity`, line-height `1.8` (`.quote-poetic`).
- **Body / Metadata**: `Poppins`, clean, uppercase tracking for dates and locations.

## UI Elements
- **Cards**: `flat` and `soft` variants with subtle border accents (`border-primary/20`) to eliminate glassmorphism monotony.
- **Buttons**: `solid`, `outline`, and `glass` variants with micro-interactions.
- **Shadows**: Soft, elegant dropshadows (`shadow-[0_8px_32px_rgba(199,169,127,0.06)]`).
- **Transitions**: Smooth `300ms` to `500ms` duration.
- **Border Radius**: Refined `rounded-2xl` and `rounded-3xl` corners.
