# Design System

## Colors
- **Primary**: `#C7A97F` (Champagne gold)
- **Secondary**: `#F8F4EE` (Warm ivory / Soft beige)
- **Accent**: `#2E4A3D` (Deep emerald)
- **Text**: `#1A1A1A` (Matte black)

## Typography Hierarchy
- H1: Cinzel / Cormorant Garamond, Large, uppercase or capitalized.
- H2, H3: Cinzel / Cormorant Garamond.
- Body: Poppins / Inter, clean, highly readable.

## UI Elements
- **Buttons**: Minimal, maybe using Accent or Primary color borders.
- **Shadows**: Soft, elegant dropshadows to elevate cards without looking bulky.
- **Transitions**: Smooth, 300ms to 500ms duration.
- **Border Radius**: Subtle rounded corners or completely sharp for modern look.
