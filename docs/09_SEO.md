# SEO & Social Sharing Guidelines

## Goals
- Optimize WhatsApp sharing preview
- Optimize Instagram/Facebook link preview
- Improve mobile browser appearance
- Ensure elegant metadata presentation

## Meta Title Format
Sameer & Nihala | Wedding Invitation

## Meta Description
Elegant digital wedding invitation for Mohammed Sameer Kallangadan & Nihala Jasmin KK.

## Open Graph Requirements
- OG Image Size: 1200x630
- Format: JPG/WebP
- Lightweight optimized image

## Required Meta Tags
- title
- description
- og:title
- og:description
- og:image
- theme-color

## Favicon
- Minimal elegant monogram preferred

## Performance Goal
- Lighthouse SEO score > 90
