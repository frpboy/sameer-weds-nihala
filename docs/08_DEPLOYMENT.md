# Deployment Checklist

- **Platform**: Vercel

## Vercel Deployment Flow
1. Connect GitHub repository to Vercel.
2. Set Framework Preset to Vite.
3. Configure environment variables (if any).
4. Deploy Main branch.

- Verify DNS records.
